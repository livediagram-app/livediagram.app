**Export** downloads the active tab in the format you need, whether that is an image for a doc, a PDF to print, or a file to move the tab into another diagram.

## Export formats

From the tab ellipsis menu, expand the **Content** section and choose **Export**, then pick a format:

- **PNG** — a raster image, good for pasting into docs and chat.
- **SVG** — a crisp vector image that scales to any size.
- **PDF** — a print-ready page of the tab.
- **Mermaid** — the tab as [Mermaid](https://mermaid.js.org/) flowchart text, which keeps every connection (unlike Markdown). Pick it to open an editor: tweak the text and **copy** it, or **download** a `.mmd` file. Your edits there never change the tab.
- **Markdown** — the tab's structure as a Markdown outline (a readable summary; it does not preserve every connection).
- **JSON** — a portable `.livediagram-tab.json` file that [imports](/help/tabs/import-tabs/) back with full fidelity.

Most formats download named after the tab, so exports stay easy to tell apart; Mermaid opens an editor first so you can copy or download.

## What the image exports capture

The PNG, SVG, and PDF exports show the tab as it looks, including its [isometric view](/help/palette/isometric-mode/) if that is on and the canvas background pattern, so what you download matches what you see. The Mermaid, JSON, and Markdown exports capture the underlying content rather than the picture you see.

<Tip>
  Reach for **SVG** when the image might be resized or printed large, it stays sharp at any scale,
  unlike a fixed-resolution PNG. For sharing a live, always-current view instead of a static file,
  see [Sharing and Embeds](/help/collaboration/sharing/).
</Tip>
