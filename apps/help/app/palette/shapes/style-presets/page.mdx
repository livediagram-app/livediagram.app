Sometimes you want a shape to stand out, or a set of arrows to look consistent, without fiddling with fill, border, weight, and pattern one control at a time. Style presets give you a **Presets** section in the selected-element context menu with ready-made looks you can apply in a single click. Presets is the first accordion inside the element's **Style** menu, alongside Colours and Border, and presets always track your active theme so they stay on-brand.

## Shape presets

Right-click a single shape, open the **Style** menu, and expand **Presets** to find a grid of ready-made looks, plus a reset. Each preset is a complete look: it sets the shape's fill, border, and text colour together with a matching border treatment (weight, pattern, and corner radius), so one click restyles the whole shape.

The grid offers up to twelve theme-derived looks, spanning the on-theme default plus soft, tinted, solid, bold, outline, muted, inked, pill, dotted, frame, and ghost variants. On multi-colour themes you also get a card per branch hue. Text colour is auto-contrasted on filled variants so labels stay readable. **Reset to default** clears the shape's colour and border overrides, returning it to its theme default in one click.

## Arrow presets

Right-click a single arrow and the **Presets** category offers eight one-click arrow styles. Each combines a line pattern (solid / dashed / dotted), a thickness, and an optional flow animation, so you get looks like a dashed animated arrow or a travelling-dot arrow. **Reset to default** clears the arrow's pattern, thickness, and animation overrides.

## Good to know

- The Presets category appears only when a single element is selected, and only the matching set shows: shape presets for a shape, arrow presets for an arrow.
- Each preset applies as one undo step, so you can revert a look in a single press.
- Because colour presets are theme-derived, switching the diagram's theme refreshes the available colours to match.
